## ✨ Features

- **🎓 Intelligent Aptitude Quiz:** A dynamic 15-question quiz to determine your strongest academic and professional stream.
- **🗺️ Course-to-Career Maps:** Deep, visually stunning bento-box maps exploring Science, Commerce, Arts, and Vocational trades.
- **🏫 College Directory:** A searchable directory of 30+ top universities and technical institutes across Chhattisgarh and India.
- **⏳ Interactive Timeline:** A roadmap from Class 10th to post-graduation showing you exactly what exams and actions to take and when.
- **📱 PWA & Offline Ready:** Built to work beautifully on mobile devices, even with patchy internet connections.
- **🔐 Secure Authentication:** Seamless user profiles powered by Supabase with persistent sessions and avatar uploads.

## 🛠️ Tech Stack

- **Frontend Toolkit:** React 18, Vite, TypeScript
- **Styling & UI:** Tailwind CSS, Shadcn UI, Framer Motion, Lucide Icons
- **Backend & Database:** Supabase (Auth, Postgres, Storage)
- **Deployment:** Vercel / Netlify (Frontend), Render (Backup API)

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js and npm installed.
You will also need a [Supabase](https://supabase.com) project with Authentication and a `profiles` table set up.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Nikhil-Vzo/Guidely-.git
   cd Guidely-
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   Create a `.env` file in the root directory:
   ```env
   VITE_SUPABASE_URL=your_supabase_project_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```
